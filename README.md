# 김종국 영어 학습법

한글 문장을 보고 영어 소리를 떠올려 암기를 확인하는 웹 앱입니다.

## 주요 기능

- **문장 관리**: 한글/영어 문장 쌍 추가, 수정, 삭제
- **엑셀 업로드**: A열(한글), B열(영어)로 구성된 엑셀 파일 일괄 업로드
- **암기 테스트**: 한글 문장을 보고 영어로 떠올린 후 셀프 체크
- **틀린 문장 복습**: 틀린 문장만 모아서 다시 테스트
- **휴지통**: 삭제한 문장 복원 가능
- **데이터 저장**: 브라우저 LocalStorage에 자동 저장

## 사용 방법

1. 한글 문장과 영어 문장을 입력하고 "추가" 버튼 클릭
2. 또는 엑셀 파일(.xlsx)을 업로드하여 일괄 추가
3. "암기 테스트 시작" 버튼으로 테스트 시작
4. 한글 문장을 보고 영어로 떠올린 후 "정답 보기" 클릭
5. 스스로 맞췄으면 "알았어요", 틀렸으면 "모르겠어요" 클릭
6. 결과 화면에서 틀린 문장만 다시 테스트 가능

## 키보드 단축키

| 키 | 동작 |
|----|------|
| `Space` / `Enter` | 정답 보기 |
| `O` | 알았어요 |
| `X` | 모르겠어요 |
| `Esc` | 종료 / 모달 닫기 |

## 기술 스택

- HTML5
- CSS3 (반응형 디자인)
- Vanilla JavaScript
- LocalStorage (데이터 저장)
- SheetJS (엑셀 파일 파싱)
